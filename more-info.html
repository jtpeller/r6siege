What is this?

If you've ever played Siege, it may sometimes be hard to figure out what operator you want to play. This website contains a couple of customizable randomizers where you could randomly select an operator or a gun meeting certain criteria.

You can get started by using the navigation bar above!

How do I use this?

Want to select a random pistol to use? What about some random primary?
  • Simple: Go here
What about selecting some random defender or attacker?
  • Easy: Click here
Why? Siege already has a randomizer!
  • Sure, Siege does have a randomizer, but it isn't customizable (nor can you randomize which gun to use). This site gives you the ability to add certain criteria to your randomization, to customize what pool of operators or guns you want to select from.
